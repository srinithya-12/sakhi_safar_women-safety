# Sakhi Safar

Sakhi Safar is a web application designed to help users travel safely, especially women. It provides basic features like registration, login, and safety support.

## Features
- User Registration
- User Login
- Simple and secure system

## Technologies Used
- HTML, CSS, JavaScript
- Flask (Python)
- MySQL

## How to Run
1. Run app.py
2. Open browser
3. Go to http://127.0.0.1:5000/
